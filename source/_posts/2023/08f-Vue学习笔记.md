## Vue 组件

vue 作为扩展名的文件是 Vue 组件，有三个部分组成：

 - template 定义模板的内容、数据渲染和 UI 组件等；
 - script 页面业务逻辑；
 - style 页面样式；
## 开发环境

使用 VS Code 作为开发环境。
